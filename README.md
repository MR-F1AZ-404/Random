## <b>installation</b>

- `pkg update`
- `pkg upgrade`
- `pkg install git`
- `pkg install python`
- `pip install requests`
- `pip install mechanize`
- `pip install bs4`
- `rm -rf Random`
- `git clone --depth=1 https://github.com/MR-F1AZ-404/Random.git`
- `cd Random`
- `python Cyber.py`
